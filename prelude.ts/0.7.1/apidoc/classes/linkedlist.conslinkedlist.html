Class ConsLinkedList<T>

ConsLinkedList holds a value and a pointer to a next element, which could be ConsLinkedList or EmptyLinkedList. A ConsLinkedList is basically a non-empty linked list. It will contain at least one element. "static methods" available through LinkedListStatic
